# [항해99] 코딩 테스트 스터디 DAY13- 그리디

<br>

# 문제. 토마툐
> [링크](https://www.acmicpc.net/problem/27961)



## 풀이

```python
import sys
import math
input = sys.stdin.readline

# 입력
n = int(input())

if n == 0:
    print(0)
elif n == 1:
    print(1)
else:
    # log2
    res = math.ceil(round(math.log(n, 2), 10))
    print(res+1)
```
